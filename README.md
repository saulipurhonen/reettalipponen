# reettapurhonen.art

Personal portfolio website for Reetta Purhonen, a Finnish visual artist based in Lappeenranta.

Built with [Astro](https://astro.build/), [TinaCMS](https://tina.io/), and [Tailwind CSS](https://tailwindcss.com/). Deployed to GitHub Pages.

## Development

```bash
cd v3
npm install
npm run dev
```

## Build & Preview

```bash
npm run build
npm run preview
```
